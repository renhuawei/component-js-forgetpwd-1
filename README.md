# 功能介绍
1. 输入用户名(手机号)，发送验证码校验用户身份；
2. 用户输入新旧密码更改密码；

# 安装方法
cmd进入工程根目录路径下(例如:HelloWorld)，执行 pastry bake bower install component-js-forgetpwd命令
